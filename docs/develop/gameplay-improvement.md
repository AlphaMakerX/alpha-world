# 玩法改进分析

## 当前优势

- Adam 央行 + Bot1 国企双层调控，保证流动性
- 三级生产链层次清晰
- 收购站成本价 ×1.2 为新手提供保底收益

## 潜在问题

- **经济单向流动**：Adam 会逐渐回收所有货币，造成通缩
- **Bot1 收购单静态**：20% 加价被摸透后变成无脑刷

## 提升方向

### 1. 动态市场

- Bot1 收购价随供需浮动
- 原材料采购价格周期波动
- 交易税，资金回流 Adam 再通过 Bot 注入

### 2. 丰富目标

- 勋章分级（青铜/白银/黄金），解锁特权
- 赛季排行榜
- 全服公共工程（共同投入解锁新内容）

### 3. 激活体力

- 生产消耗体力，住宅恢复体力
- 形成工厂+住宅配比决策

### 4. Agent 进阶

- 多 Bot 角色分工（原材料/成品/基建）
- Agent 根据市场数据动态调策略
- 央行货币政策（通胀提高费用，通缩发补贴）
