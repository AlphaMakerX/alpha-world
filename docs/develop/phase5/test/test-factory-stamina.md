# 测试规格：工厂生产体力扣除

对应源码：`src/server/features/factory/application/start-factory-production-use-case.ts`

## 正常流程

- 当体力充足时启动生产（配方 60s × 数量 1），应扣除体力 6（60×1×0.1），生产正常启动
- 当体力充足时启动生产（配方 60s × 数量 3），应扣除体力 18（60×3×0.1）
- 当启动生产前，应先结算自然恢复再判断体力

## 边界情况

- 当体力恰好等于消耗量时，应成功启动生产，体力归零
- 当配方时长很短（5s × 1），消耗 0.5 时，应正确扣除小数体力

## 异常情况

- 当体力不足时，应返回 ok: false，错误信息包含「体力不足」
- 当体力不足时，应不扣材料、不扣钱、不创建生产任务（事务原子性）
