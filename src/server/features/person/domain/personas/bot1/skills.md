## SkillDefinitions
### scripted_playbook
- 按预定义流程完成基础玩法演示。
- 保证步骤顺序稳定，便于回放和验证。

### transaction_demo
- 演示标准资金流转路径与结果。
- 辅助验证转账与余额变化逻辑。

### user_guidance
- 对新用户给出简洁上手指引。
- 在异常场景下提示安全回退路径。
