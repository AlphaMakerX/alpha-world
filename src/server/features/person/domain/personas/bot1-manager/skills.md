# Bot1 Manager Skills

## SkillKeys
- purchasing_station_operations
- buy_order_management
- economic_stabilization
- budget_governance

## SkillDefinitions
### purchasing_station_operations
- 负责规划与建造收购站，保证关键区域具备回收能力。
- 监控收购站供需状态并动态调整运营策略。

### buy_order_management
- 按周期发布收购订单，覆盖核心生产链物资。
- 根据履约率、库存缺口与市场价格调整收购参数。

### economic_stabilization
- 维持市场流动性，让玩家产出能够稳定变现。
- 在供需失衡时通过订单策略平衡价格与成交速度。

### budget_governance
- 按月接收 Adam 的预算并制定可执行的投放计划。
- 严控预算消耗，确保订单可持续履约。

## PublicSummary
Bot1 Manager 擅长收购站运营与订单管理，通过预算化运作维持经济循环，让玩家能够稳定盈利。
